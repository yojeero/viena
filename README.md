### [Viena](https://viena.vercel.app/) is a Vanilla radio player.   
  
- Instant one-click 5 radio playback   
- Responsive & retina-ready   
- Tailwind v4   
- Vanilla JS powered    
- Customize stations   
- Live audio visualizer   
- Full keyboard control   
- Easy station switching   
- Built-in live clock   
- Fast and lightweight

#### Radio player features    

#### Instant Radio Playback      

One-click play and seamless station switching   
Built for smooth, uninterrupted listening   

#### Immersive Audio Visuals   

Real-time animated sound bars that react to the music   
Clean, symmetrical visualizer for a modern radio feel   
Subtle idle animation even when paused   

#### Keyboard-First Controls   

Control playback without touching the mouse   
Space to play/pause, arrows to change stations   
Perfect for multitasking and desktop listening   

#### Multiple Stations, One Player   

Switch easily between radio stations or tracks   
Auto-plays the next station when a stream ends   
Remembers your last choice automatically

#### Live Clock Interface   

Always-visible current time and day   
Adds a real radio / dashboard vibe   

#### Smooth & Responsive Design   

Adapts to all screen sizes   
Crisp visuals on high-resolution displays   
Automatically adjusts when the window is resized   

#### Fast & Lightweight   

No heavy dependencies   
Optimized for performance and low CPU usage   
Starts instantly without lag   

#### Modern Web Technology   

Powered by native browser audio capabilities   
No plugins or external players required   

#### Perfect for Web Radio Projects   

Ideal for personal radio sites   
Great for streaming dashboards and music apps   
Easy to customize and extend
